# Код детям 🎮

Курс программирования для детей 8–9 лет — год обучения через сборку своих игр.
Ребёнок пишет **настоящий код** (JavaScript), но начинает с «покрути ручки»: меняет числа и слова в готовом коде и сразу видит результат. Каждый урок даёт одну победу, которую можно показать.

Работает **офлайн, без интернета и без установки** — просто открой файл в браузере.

## Как запустить

1. Скачай проект (кнопка **Code → Download ZIP** на GitHub) и распакуй.
2. Открой **`index.html`** двойным кликом — откроется главная страница с программой курса.
3. Выбирай урок и занимайся. Прогресс сохраняется автоматически в браузере.

Подойдёт любой современный браузер: Chrome, Edge, Яндекс, Firefox.

## Как это устроено

- `index.html` — главная: вся программа курса и твой прогресс.
- `lessons/` — уроки (по одному файлу на урок).
- `shared/` — общий код: движок (`engine.js`), каркас урока (`lesson.js`), стили (`styles.css`) и список всех уроков (`curriculum.js`).

Прогресс хранится в браузере (localStorage) и привязан к тому, **где лежит папка** с проектом.

## Как обновиться (когда выйдут новые уроки)

**Просто:**
1. На главной странице нажми **«Сохранить прогресс»** — скачается файл `progress.json`.
2. Скачай новый архив (кнопка появится на главной, когда выйдет обновление).
3. Распакуй его **поверх старой папки, заменив файлы** (Windows спросит «Заменить?» — да).
   ⚠️ Не распаковывай рядом — иначе получится вторая копия, и прогресс может «потеряться».
4. Открой `index.html`. Прогресс на месте. Если вдруг нет — нажми **«Загрузить прогресс»** и выбери сохранённый `progress.json`.

**Для тех, кто умеет git:**
```
git pull
```
в папке проекта — обновит файлы на месте, прогресс сохранится сам (папка та же).

Файл `progress.json` — это ещё и парашют: им можно перенести прогресс на другой компьютер или в другой браузер (кнопка «Загрузить прогресс»).
